# imethan-blog-2
一个自己动手撸出来的技术博客，用于编写Markdown风格的技术内容，imethan-blog的升价版本；后端基于Springboot、Spring Security、Guava、thymeleaf、Sitemesh3、MongoDB；前端基于JQuery、Bootstrap、Editor.md开发，下一步计划用Vue重写前端实现。

访问地址：www.imethan.cn
